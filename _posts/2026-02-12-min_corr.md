# Minimum correlation

Consider we trade a single asset returns $y$ based on some features $x$. Optimal weight is

$w(x) = \frac{\nu\_{y\|x}}{\sigma\_{y\|x}^2}$

This weight induces a strategy $s = wy$ with expected value:

$\mathbf{E} \left\[ s \right\] = \mathbf{E}\_x\left\[ \frac{\mu\_{y\|x}^2}{\sigma\_{y\|x}^2} \right\]$

Now, let's assume that variance is constant and small correlations between feature and target; this makes the conditional variance approximate to target variance (think of a conditional gaussian for example). Under this:

$\mathbf{E} \left\[ s \right\] \approx \frac{\mathbf{E}\_x \left\[ \mu\_{y\|x}^2\right\]}{\sigma^2} = \frac{\text{Var}\left(\mathbf{E}\left\[y\|x\right\]\right)}{\text{Var}\left\[y\right\]}$

As previously seen in other posts, this quantity can be identified as the explained variance ratio. For a linear model this is equal to correlation squared $\rho^2$ between feature and target. It represents the fraction of $y$ variance that can be explained with $x$. So:

$\mathbf{E} \left\[ s \right\] \approx \rho^2$

If, for each trade, we pay a fee $c$ (in percentage), we need to discount to the expected value the product of $c$ with weight to get a more correct measure of expected strategy return. We can use the standard deviation of weights as a measure:

$\sqrt{\mathbf{E}\left\[w^2\right\]} = \sqrt{\mathbf{E}\left\[\frac{\mu\_{y\|x}^2}{\sigma\_{y\|x}^4}\right\]} \approx \sqrt{\frac{\mathbf{E}\left\[\mu\_{y\|x}^2\right\]}{\sigma^4}} = \frac{\|\rho\|}{\sigma}$

So a refined expected return is (we could have solved the optimal allocation with the cost constraint but here the idea is to make a simpler estimator):

$\mathbf{E} \left\[ s \right\] = \rho^2 - c \frac{\|\rho\|}{\sigma}$

and $\mathbf{E} \left\[ s \right\] > 0$ implies

$\|\rho\| > \frac{c}{\sigma}$

This formula is a simple and good plugin estimator to what is a good correlation; also, we can interpret it as a measure of the order of magnitude of correlations that are easy to find (as those signals are not profitable).
