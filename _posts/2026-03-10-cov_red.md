# A method to regularize covariance





Assuming a zero-mean multivariate normal distribution $p(x\|\Sigma) \sim \mathcal{N}(0, \Sigma)$, the likelihood for data $X$ with $n$ observations is:

$p(X\|\Sigma) = (2\pi)^{-\frac{np}{2}} \det(\Sigma)^{-n/2} \exp \left( -\frac{1}{2}\text{tr}(S\Sigma^{-1})\right)$

where $S = \sum_i x_i x_i^T$ is the scatter matrix and $\hat{\Sigma} = \frac{1}{n}S$ is the sample covariance.




By placing an Inverse-Wishart (IW) prior on the covariance:

$p(\Sigma\|\nu, \Psi) \propto \det(\Psi)^{\frac{\nu}{2}} \det(\Sigma)^{-\frac{\nu+p+1}{2}} \exp \left( -\frac{1}{2} \text{tr} (\Psi \Sigma^{-1})\right)$ 

The resulting posterior is $p(\Sigma\|X) \sim IW(\nu+n, \Psi+S)$. The posterior mean provides a natural shrinkage estimator as a convex combination of the sample covariance and the prior mean $\hat{\Sigma}_0$:

$\mathbb{E}[\Sigma\|X] = \frac{S+\Psi}{n+\nu-p-1} = (1-\alpha) \hat{\Sigma} + \alpha \hat{\Sigma}_0$

As the mean of a IW is given by $\mathbb{E}[\Sigma] = \frac{\Psi}{\nu - p - 1}$

Conditioned on the prior parameters, we can write the data likelihood as

$p(X\|\nu, \Psi) = \int p(X\|\Sigma)p(\Sigma) \text{d}\Sigma = \pi^{-\frac{np}{2}} \frac{\Gamma_p \left(\frac{\nu+n}{2}\right)}{\Gamma_p \left(\frac{\nu}{2}\right)}\frac{\det(\Psi)^{\frac{\nu}{2}}}{\det(\Psi + S)^{\frac{\nu+n}{2}}}$


### Unconstrained optimization

If $\Psi$ is any positive definite matrix, $\nabla_{\Psi}L = 0$ gives:

$\nu \Psi^{-1} = (\nu + n)(\Psi + S)^{-1} \rightarrow \Psi^{*} = \frac{\nu}{n} S$

The evidence-maximizing scale matrix is proportional to the sample scatter matrix. Being the posterior mean a convex combination of prior and sample covariance, we get:

$\mathbb{E}[\Sigma\|X] = \hat{\Sigma}$

We need to impose some structure in order for something interesting to show up.



### Isotropic

Let $\Psi = \lambda I$. With that  $\det(\Psi) = \det(\lambda I) = \lambda^p$ and $\det(\Psi + S) = \det(\lambda I + S) = \Pi_i (\lambda + d_i)$ with $d_i$ the eigenvalues of $S$ (use $S = UDU^T$). From the previous expression we can write the log likelihood:

$L = \log p(X\|\nu, \lambda) \propto \log \left( \frac{\Gamma_p \left(\frac{\nu+n}{2}\right)}{\Gamma_p \left(\frac{\nu}{2}\right)} \right) + \frac{\nu p}{2} \log(\lambda) - \frac{\nu+n}{2} \sum_i \log(\lambda + d_i)$

Objective is to find optimal parameters $\nu, \lambda$ that maximize $L$. This condition can be expressed as

$\frac{\partial L}{\partial \nu} = 0 \rightarrow f_1(\lambda, \nu) = 0 \rightarrow \psi_p\left(\frac{\nu+n}{2}\right) - \psi_p\left(\frac{\nu}{2}\right) + p\log(\lambda) - \sum_i \log(\lambda+d_i) = 0$

$\frac{\partial L}{\partial \lambda} = 0 \rightarrow f_2(\lambda, \nu) = 0 \rightarrow \frac{\nu p}{\lambda}-(\nu+n)\sum_i \frac{1}{\lambda+d_i} = 0$



with $\psi_p(a) = \sum_{j=1}^p \psi(a+\frac{1-j}{2})$ and $\psi(a)$ the digamma function. Also, $\lambda>0$ and $\nu>p+1$.

#### Newton method

Find roots of the system of equations. Solve for each variable independently and iterate (block coordinate). Update solution like

$x_{k+1} = x_k - \delta \frac{f(x_k)}{f'(x_k)}$

We can adjust $\delta$ over the iteration to make sure we are within the bounds for each variable. 

With $\nu$ frozen, we want to solve $\frac{\partial L}{\partial \lambda} = 0$ so we need to to the update

$\lambda_{k+1} = \lambda_k - \delta_{\lambda, k} \frac{f_2(\lambda_k, \nu)}{\frac{\partial f_2}{\partial \lambda}}$

Then, with $\lambda$ frozen, we want to solve $\frac{\partial L}{\partial \nu} = 0$ so we need to to the update

$\nu_{k+1} = \nu_k - \delta_{\nu, k} \frac{f_1(\nu_k, \lambda)}{\frac{\partial f_1}{\partial \nu}}$

This system iterates into the roots.

Also:

$\frac{\partial f_1}{\partial \nu} = \frac{1}{2}\left( \psi_p' \left(\frac{\nu+n}{2}\right) - \psi_p' \left(\frac{\nu}{2}\right)\right)$

$\frac{\partial f_2}{\partial \lambda} = -\frac{\nu p}{\lambda^2} + (\nu+n)\sum_i\frac{1}{(\lambda+d_i)^2}$


and $\psi_p' = \sum_{j = 1}^p \psi_1(a+\frac{1-j}{2})$ and $\psi_1$ the trigamma function.


Below is a implementation of this.



```python
import numpy as np
from scipy.special import psi, polygamma

# special function implementation, this gives the same as using scipy
def _digamma(x: float) -> float:
    if x <= 0.0:
        raise ValueError("digamma approximation requires x > 0")

    out = 0.0
    while x < 8.0:
        out -= 1.0 / x
        x += 1.0

    inv = 1.0 / x
    inv2 = inv * inv
    out += (
        np.log(x)
        - 0.5 * inv
        - inv2 / 12.0
        + inv2 * inv2 / 120.0
        - inv2 * inv2 * inv2 / 252.0
    )
    return out


def _trigamma(x: float) -> float:
    if x <= 0.0:
        raise ValueError("trigamma approximation requires x > 0")

    out = 0.0
    while x < 8.0:
        out += 1.0 / (x * x)
        x += 1.0

    inv = 1.0 / x
    inv2 = inv * inv
    inv3 = inv2 * inv
    inv5 = inv3 * inv2
    inv7 = inv5 * inv2
    inv9 = inv7 * inv2

    out += (
        inv
        + 0.5 * inv2
        + inv3 / 6.0
        - inv5 / 30.0
        + inv7 / 42.0
        - inv9 / 30.0
    )
    return out

def _multivariate_digamma(a: float, p: int) -> float:
    return sum(_digamma(a + (1.0 - j) / 2.0) for j in range(1, p + 1))

def _multivariate_trigamma(a: float, p: int) -> float:
    return sum(_trigamma(a + (1.0 - j) / 2.0) for j in range(1, p + 1))

def fit_isotropic_iw(X, center=True, n_iter=100, tol=1e-8):
    n, p = X.shape
    if center:
        X = X - X.mean(axis=0)

    # Precompute eigenvalues (The only heavy calculations)
    d = np.linalg.eigvalsh(X.T @ X)
    d = np.maximum(d, 1e-10) # clip eigenvalues to small threshold
    
    # Initialization
    tau0 = np.trace(X.T @ X) / (n * p)
    nu = p + 10.0
    lam = nu * tau0
    j_range = np.arange(p)

    for it in range(n_iter):
        
        # 1
        f2 = (nu * p) / lam - (nu + n) * np.sum(1.0 / (lam + d))
        f2_lam = -(nu * p) / (lam**2) + (nu + n) * np.sum(1.0 / (lam + d)**2)

        step_lam = -f2 / f2_lam
        # Damp to ensure positive scale
        s = 1.0
        while lam + s * step_lam <= 0: 
            s *= 0.5
        lam += s * step_lam

        # 2
        # uncomment to use the scipy functions
        # mv_psi = np.sum(psi(0.5 * (nu + n - j_range))) - np.sum(psi(0.5 * (nu - j_range)))
        # mv_tri = np.sum(polygamma(1, 0.5 * (nu + n - j_range))) - np.sum(polygamma(1, 0.5 * (nu - j_range)))
        mv_psi = _multivariate_digamma(0.5*(nu+n), p) - _multivariate_digamma(nu*0.5, p) 
        mv_tri = _multivariate_trigamma(0.5*(nu+n), p) - _multivariate_trigamma(nu*0.5, p)

        f1 = mv_psi + p * np.log(lam) - np.sum(np.log(lam + d))
        f1_nu = 0.5 * mv_tri
        
        step_nu = - f1 / f1_nu
        # Damp to maintain proper distribution
        s_nu = 1.0
        while nu + s_nu * step_nu <= (p - 1 + 1e-5): 
            s_nu *= 0.5
        nu += s_nu * step_nu
        
        # Check convergence on both updates
        if max(abs(s * step_lam), abs(s_nu * step_nu)) < tol:
            break
    
    return {
        "nu": nu,
        "lam": lam,
        "converged": it < n_iter - 1,
        "iters": it + 1
    }
```

## Ledoit-Wolf

The proposed method is quite similar to Ledoit-Wolf shrinkage. Let us present it here. 

Define the estimator as a convex combination of the sample covariance ($S$) and a structured target ($F = \mu I$):

$\Sigma^* = (1-\alpha)S + \alpha F$

The objective is to minimize the risk $R(\alpha)$:

$R(\alpha) = \mathbb{E}[\\|\Sigma^* - \Sigma\\|_F^2]$

where $\\| \cdot \\|_F$ is the Frobenius norm. 

We can rewrite the error term as:$$\Sigma^* - \Sigma = (S - \Sigma) + \alpha(F - S)$$

Recall that for any matrix $A$, $\\|A\\|_F^2 = \langle A, A \rangle = \text{tr}(A A^T)$. Using the linearity of expectation and the properties of the Frobenius inner product:

$R(\alpha) = \mathbb{E}[\\|S - \Sigma\\|^2] + \alpha^2 \mathbb{E}[\\|F - S\\|^2] + 2\alpha \mathbb{E}[\langle S - \Sigma, F - S \rangle]$

To find the minimum, we differentiate with respect to $\alpha$ and set to zero: $\frac{dR}{d\alpha} = 2\alpha \mathbb{E}\[\\|F - S\\|^2] + 2\mathbb{E}\[\langle S - \Sigma, F - S \rangle] = 0$ Solving for $\alpha$ gives:

$\alpha^* = \frac{\mathbb{E}\[\langle \Sigma - S, F - S \rangle]}{\mathbb{E}\[\\|F - S\\|^2]}$


Let's look at the term $\mathbb{E}[\langle \Sigma - S, F - S \rangle]$. Since $F = \mu I$, and $\mathbb{E}[S] = \Sigma$, we expand the inner product:

$\langle \Sigma - S, F - S \rangle = \text{tr}((\Sigma - S)(F - S)) = \text{tr}(\Sigma F - \Sigma S - S F + S^2)$

Taking the expectation:

$\mathbb{E}\[\dots] = \text{tr}(\Sigma F) - \text{tr}(\Sigma \mathbb{E}\[S]) - \text{tr}(\mathbb{E}[S] F) + \text{tr}(\mathbb{E}\[S^2]) = \text{tr}(\Sigma F) - \text{tr}(\Sigma^2) - \text{tr}(\Sigma F) + \text{tr}(\mathbb{E}\[S^2]) = \text{tr}(\mathbb{E}\[S^2] - \Sigma^2)$

By the definition of variance ($Var(X) = \mathbb{E}\[X^2] - \mathbb{E}\[X]^2$), this is the sum of the variances of all entries in $S$: $\beta^2 = \sum_{i=1}^p \sum_{j=1}^p Var(s_{ij}) = \mathbb{E}\[\\|S - \Sigma\\|_F^2]$


Because $\Sigma$ is unknown, we cannot calculate $\beta^2$ directly. Let $x_i$ be the $i$-th observation (a $p \times 1$ vector). The sample covariance is $S = \frac{1}{n} \sum_{i=1}^n x_i x_i^T$. The variance of the average of $n$ i.i.d. variables is the variance of one variable divided by $n$: 

$\beta^2 \approx \frac{1}{n^2} \sum_{i=1}^n \\|x_i x_i^T - S\\|_F^2$

We can use the trace identity: 

$\\|x_i x_i^T - S\\|_F^2 = \\|x_i\\|_2^4 - 2\\|x_i S x_i^T\\| + \\|S\\|_F^2$

For a more efficient way to compute (vectorize operations).

The optimal shrinkage intensity is $\alpha = \frac{\hat{\beta}^2}{\hat{\delta}^2}$, where:

- The Target ($\mu$): The value that minimizes $\\|S - \mu I\\|^2 \rightarrow \mu = \frac{\text{tr}(S)}{p}$
- The Misspecification ($\delta^2$): How far the sample covariance is from the target. $\delta^2 = \\|S - \mu I\\|_F^2$
- The Estimation Error ($\beta^2$): The variance of the sample covariance entries. $\beta^2 = \frac{1}{n^2} \sum_{i=1}^n \\|x_i x_i^T - S\\|_F^2$
 
If $\beta^2$ is large (high noise), $\alpha$ moves toward 1 (more shrinkage). If $\delta^2$ is large (the data is very non-isotropic), $\alpha$ moves toward 0 (less shrinkage).

Below is a implementation of this.


```python
def ledoit_wolf_shrinkage(X, center = True):
    """
    Assumes X is centered.
    """
    n, p = X.shape
    if center:
        X = X - X.mean(axis=0)
    S = (X.T @ X) / n
    
    # Target: mu * I
    mu = np.trace(S) / p
    
    # Delta squared: Squared Frobenius norm ||S - mu*I||^2
    # Equivalent to sum(S^2) - p*(mu^2)
    delta_sq = np.sum(S**2) - p * (mu**2)
    
    # Beta squared: Simplified consistent estimator
    # We use: beta_sq = (1/n^2) * sum(||xi.T @ xi - S||^2)
    # Optimized: (1/n^2) * sum(||xi||^4) - (1/n) * ||S||^2
    # This avoids forming n separate p x p matrices.
    X_sq = X**2
    norm_xi_sq = np.sum(X_sq, axis=1) # ||xi||^2 for each row
    beta_sq = (np.sum(norm_xi_sq**2) / n**2) - (np.sum(S**2) / n)
    
    # Clamped shrinkage intensity
    # Ensure denominator is not zero (for p=1 or degenerate cases)
    if delta_sq == 0:
        alpha = 1.0
    else:
        alpha = np.clip(beta_sq / delta_sq, 0, 1)
    
    # Return the convex combination
    # S_shrink = (1 - alpha) * S + alpha * mu * np.eye(p)
    return {
        "alpha": alpha,
        "mu": mu
    }
```

Let us make some tests to compare the outputs of the methods. I will not enter into discussion which one is better as for Gaussian variables they should tend to the same parameters. For our method we can compute an _equivalent_ $\alpha = \frac{\nu-p-1}{n+\nu-p-1}$ and compare with the one from LW.





```python
from scipy.stats import multivariate_normal

def generate_test_data(n=50, p=100, k = 5):
    """ Generate high-dimensional data (p > n) with a structured covariance """
    # True covariance: Identity + a few strong correlations (Spiked Model)
    true_cov = np.eye(p)
    true_cov[0:k, 0:k] += 5.0  # Top-left block has high variance
    # Generate zero-mean data
    X = np.random.multivariate_normal(np.zeros(p), true_cov, size=n)
    return X, true_cov

```
